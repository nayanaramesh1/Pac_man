Pac-Man AI Game using Pygame

This project is an AI-based implementation of the classic Pac-Man game developed using Python and Pygame. The game features a dynamically generated maze, interactive gameplay, and an intelligent ghost powered by the A* search algorithm.

The ghost acts as an AI agent that uses heuristic-based pathfinding (Manhattan distance with stochastic variation) to chase the player when nearby and exhibit random wandering behavior when far away. This creates a balanced and realistic gameplay experience.

The game also includes multiple levels with increasing difficulty, collectible coins and special coins, score tracking, and collision detection.
